# HABITS BOT

Bot for tracking habits via telegram API
Data storage: firebase

## Environments

Add `.env` file with following data:

```js
// Bot token
BOT_TOKEN=

//  Firebase
API_KEY=
AUTH_DOMAIN=.firebaseapp.com
DATABASE_URL=https://firebasedatabase.app
PROJECT_ID=
STORAGE_BUCKET=
MESSAGING_SENDER_ID=
APP_ID=
MEASUREMENT_ID=
```
